¿Qué hace este bot?

Bot automático para Minecraft que puede:

  • • Minar automáticamente con IA
  • • Recolectar recursos específicos
  • • Navegar por el mundo
  • • Gestionar inventario automáticamente
  • • Auto-alimentación y supervivencia
  • • Comandos de agricultura

Instalación y Configuración

Windows

1. Descargar Node.js desde: nodejs.org

2. Instalar siguiendo el asistente

3. Reiniciar la computadora

Linux

Ubuntu/Debian:

sudo apt update && sudo apt install nodejs npm git

Arch Linux:

sudo pacman -S nodejs npm git

Paso 1: Descargar el Bot

Clona el repositorio:

git clone https://github.com/JDamianCabello/BotMinecraft-Node.git

Entra a la carpeta:

cd BotMinecraft-Node

Paso 2: Instalar Dependencias

npm install

Paso 3: Configurar el Bot

Edita el archivo config.json con estos valores:

{ "host": "frogcraft.org", "port": 25565, "username": "NombreDelBot", "version": "1.21", "auth": "microsoft", // o "offline" si es no-premium "commander": "TuNombreDeUsuario", // ¡IMPORTANTE! "seed": "-6207892147998764319" }
Configuración del Commander:
Cambia "TuNombreDeUsuario" por tu nombre exacto en Minecraft. Solo tú podrás controlar el bot.

Paso 4: Iniciar el Bot

node bot-modular.js
Si todo está bien configurado, verás:
- "Bot conectado a frogcraft.org"
- "Logueado como [NombreDelBot]"
- El bot aparecerá en el servidor

Comandos Disponibles

Todos los comandos disponibles con /help en el juego:

NAVEGACIÓN

/goto, /ir
Ir a coordenadas spawn (-2,12,39)
/reset
Ir al lobby y salir del spawn
/tpme
Teleportarse a v0xxii
Auto-acepta teleportes de v0xxii

MINERÍA

/mine
Iniciar minería automática inteligente
/stopmine
Detener minería
/mineobsidiana
Minar obsidiana en pilares del End
Siempre mira abajo
/stopmineobsidiana
Detener minería de obsidiana
/ores
Ver tabla de Y levels óptimos para cada ore
/collect <tipo>
Recolectar bloque específico
Ej: /collect stone
/collectall <tipo> [cantidad]
Recolectar múltiples bloques
Ej: /collectall stone 10

INVENTARIO

/drop
Tirar todos los items al suelo
/scan
Escanear bloques cercanos
/inv
Ver inventario del bot
Prioriza: diamante > esmeralda > ancient debris > oro > hierro
Evita lava automáticamente
Se cierra automáticamente cuando está lleno

ALIMENTACIÓN

/eat
Comer inmediatamente
/autofeed
Activar/desactivar auto-alimentación

AGRICULTURA

/farm
Recolectar crops automáticamente
/plant <tipo>
Plantar semillas específicas
